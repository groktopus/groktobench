## Prompt

Research arXiv for recent papers on embedding compression techniques (like Matryoshka Representation Learning or adaptive dimensionality reduction). Based on the findings, create an implementation plan. Then implement a minimal Python prototype that demonstrates the approach on a small embedding dataset. Evaluate the quality using cosine similarity comparison between compressed and uncompressed embeddings. Finally, inspect your codebase to report file structure and key metrics.

Start with research, then plan, then implement, then evaluate, then inspect. Use the appropriate skill for each step and carry findings forward between steps.
